What it does

snip add Save a new snippet with a name, language, and tags.
Content comes from stdin or your $EDITOR.
snip find ★ Fuzzy-search your entire library in real time.
Press enter to copy. Placeholder vars resolve automatically.
snip copy Copy any snippet directly to your clipboard.
If the snippet has {{VAR}} placeholders, you'll be prompted.
snip list Print all snippets as a flat table.
Filter by --lang or --tag.
snip edit Open any snippet in your $EDITOR.
Changes save back to the database on close.
snip rm Remove a snippet by ID.
Asks for confirmation before deleting.
